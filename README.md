<h1>🛡️ SafeSphere – Tech for Women's Safety &amp; Security</h1><br>

SafeSphere is an <b>AI-powered women's safety platform</b> developed for hackathons.<br>
It empowers women by providing emergency assistance, safe navigation,<br>
AI-powered guidance, and community-driven incident reporting.<br><br>

<h2>⚙️ Installation</h2>

<b>1. Clone the repository</b><br>

<pre>
git clone https://github.com/Arkajyoti-Mallick/team_elite-v2v.git
</pre>

<b>2. Navigate to the project</b><br>

<pre>
cd SafeSphere
</pre>

<b>3. Install dependencies</b><br>

<pre>
npm install
</pre>

<b>4. Start the development server</b><br>

<pre>
npm run dev
</pre>

<hr>

<h2>🔥 Environment Variables</h2>

Create a <code>.env</code> file in the project root and add the following:<br><br>

<pre>
VITE_FIREBASE_API_KEY=

VITE_FIREBASE_AUTH_DOMAIN=

VITE_FIREBASE_PROJECT_ID=

VITE_FIREBASE_STORAGE_BUCKET=

VITE_FIREBASE_MESSAGING_SENDER_ID=

VITE_FIREBASE_APP_ID=

VITE_GOOGLE_MAPS_API_KEY=

VITE_GEMINI_API_KEY=
</pre>
